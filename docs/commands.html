Basic Commands

PRINT

Displays text or variable values

PRINT "HELLO"
PRINT X
PRINT "VALUE IS: "; X

LET

Assigns values to variables

LET X = 42
LET A$ = "HELLO"

GOTO

Jumps to a specified line number

10 PRINT "LOOP"
20 GOTO 10

Function Keys

  • F1 - HELP
  • F2 - LIST
  • F3 - GOTO
  • F4 - RUN
  • F5 - CONT
  • F6 - LIST.
  • F7 - TRON
  • F8 - TROFF
  • F9 - KEY
  • F10 - SCREEN
