Compilation Instructions

For C++ Version:

# Windows (MinGW):
g++ zuix.cpp -o zuix.exe -std=c++11

# Linux/Mac:
g++ zuix.cpp -o zuix -std=c++11

For Zig Version:

# All platforms:
zig build-exe zuix.zig

System Requirements

  • C++11 compatible compiler or Zig compiler
  • Terminal with ANSI support
  • 64K of available memory (just kidding!)
