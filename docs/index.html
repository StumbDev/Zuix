About ZUIX-DOS

ZUIX-DOS is a modern implementation of BASIC that combines features from classic BASIC implementations with modern programming capabilities. It provides a nostalgic computing experience while offering powerful features.

Quick Start

10 PRINT "HELLO WORLD"
20 GOTO 10
RUN

Features

  • Classic BASIC syntax
  • Line number based programming
  • Function key support
  • File operations
  • Graphics commands
  • Sound support
  • String manipulation
  • Mathematical functions
